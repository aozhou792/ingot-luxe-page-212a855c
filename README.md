## About

**Alibarbar Australia** is an online store dedicated to the Alibarbar Ingot 9000 range — up to **9000 puffs**, **22ml** e-liquid, **2350mAh** battery, and a smart **LED display** showing battery and e-liquid levels.

This repository contains the project source code and a full **Markdown knowledge centre** (FAQ, documentation, guides) for reference and SEO.
