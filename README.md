## About

**Alibarbar Australia** is an independent online retailer specialising in the Alibarbar Ingot 9000 range. The device is rated for up to **9000 puffs** and features **22 ml** of pre-filled e-liquid, a **2350 mAh** built-in battery, and a smart **LED display** for battery and e-liquid levels.

This repository contains:

- the Vite, React and TypeScript source for the official website;
- a public Markdown knowledge centre with guides, FAQ and specifications;
- a lightweight [GitHub Pages knowledge hub](https://aozhou792.github.io/ingot-luxe-page-212a855c/);
- machine-readable discovery files, including [`llms.txt`](https://www.ailibarbar.com/llms.txt) and the [XML sitemap](https://www.ailibarbar.com/sitemap.xml).

## Local development

```bash
npm install
npm run dev       # start the local Vite server
npm run test      # run the test suite
npm run lint      # run ESLint
npm run build     # generate and build the production site
```

Stack: **Vite · React · TypeScript · Tailwind · Vercel**
